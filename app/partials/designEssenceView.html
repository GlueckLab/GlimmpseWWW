The Design Essence Matrix

In the general linear model, $\boldsymbol{Y} = \boldsymbol{XB} + \boldsymbol{E}$, the $\boldsymbol{X}$ matrix contains predictor and covariate information. For power analysis, please specify a design essence matrix, $Es(\boldsymbol{X})$. The $Es(\boldsymbol{X})$ matrix contains one and only one copy of each unique row in the full design matrix. This allows separation of the study design information from overall and relative sample size.

Enter the $Es(\boldsymbol{X})$ matrix below. To change the row dimension of the matrix, enter the updated number of rows in the left-most textbox above the matrix data. To change the column dimension, enter the desired number of columns in the right-most textbox above the matrix data. Please use a full rank coding for this matrix.
